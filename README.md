
Interactive Excel dashboard analyzing Superstore sales, profit trends, and category performance using pivot tables and charts

Sales Data Analysis Dashboard :
This project presents an interactive Sales Data Analysis Dashboard built entirely in Microsoft Excel. It uses Excel’s advanced features such as Pivot Tables, Pivot Charts, Slicers, and Data Modeling to provide a clear and dynamic view of sales performance from 2011 to 2014.
Purpose of This Project

This dashboard was created to:
•	Practice and demonstrate data analysis skills
•	Showcase Excel-based visualization capabilities
•	Provide a tool for sales performance tracking
•	Support data-driven decision making

Project Overview :
The Excel dashboard visualizes sales data across multiple dimensions including:
•	Year
•	Region
•	Segment
•	Category
•	Sub-Category
•	Month
•	Ship Mode
Through interactive slicers and visual charts, users can filter the data easily and explore insights from different angles.

Tools Used :
•	Microsoft Excel
•	Pivot Tables
•	Pivot Charts
•	Slicers
•	Conditional Formatting
•	Dashboard Design Techniques
